
# Finance & Accounting Dashboards (Complete Project)

## Overview
End-to-end financial analytics project covering profitability, expense control, cash flow,
income statements, KPIs, and investment ROI.

## Dashboards
1. Financial Health & Profitability
2. Expense Tracking & Budget Analysis
3. Cash Flow Monitoring
4. Income Statement & KPI Analysis
5. Investment & ROI Analytics

## Tech Stack
- Power BI
- SQL
- Python (Pandas, NumPy)
- Excel / CSV

## Business Impact
- Faster financial decision-making
- Improved cash management
- Cost optimisation
- ROI-driven investment insights

## How to Use
1. Load CSV files into Power BI
2. Use SQL scripts for analysis
3. Automate KPIs using Python
4. Publish dashboards
